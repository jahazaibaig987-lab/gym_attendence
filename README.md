# 🏋️ Gym Attendance Prediction (Machine Learning Project)

This project predicts whether a gym member will attend the gym or not based on workout and membership data.

## 📊 Technologies Used

* Python
* Pandas
* Scikit-learn
* Random Forest

## 🔄 Project Pipeline

1. Data Ingestion
2. Data Cleaning
3. Feature Engineering
4. Model Training
5. Prediction

## 📁 Project Structure

* **data/** → Dataset files
* **src/** → Machine learning pipeline code
* **notebook/** → Experiments and analysis
* **artifacts/** → Saved model and processed data

## 🎯 Goal

Build a machine learning model that can predict gym attendance using workout and member information.
